# 🎬 Clone da Netflix - Imersão Front-End Alura

Projeto desenvolvido durante a **Imersão Front-End da Alura**, com o objetivo de recriar uma interface inspirada na Netflix utilizando HTML, CSS e JavaScript.

---

## 🚀 Sobre o projeto

A proposta deste projeto é construir, do zero, uma página inspirada na Netflix, aplicando conceitos fundamentais de desenvolvimento web.

Neste primeiro momento, foi desenvolvida a **estrutura da página utilizando HTML**.

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3 (próximo passo 🚧)
* JavaScript (em breve)

---

## 🎯 Funcionalidades

* Estrutura inicial da interface inspirada na Netflix
* Organização de perfis de usuários
* Base pronta para estilização com CSS

---

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

* Estruturação de páginas com HTML
* Uso de tags semânticas
* Organização inicial de layout

---

## ▶️ Como executar o projeto

```bash
# Clone o repositório
git clone https://github.com/letxiz/imersao-alura-front.git

# Acesse a pasta
cd Netflix

# Abra no navegador
index.html
```
